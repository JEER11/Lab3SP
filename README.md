# HiFive1 LED Control Demo

This project is a PlatformIO-based embedded system application for the SiFive HiFive1 Rev B RISC-V development board.

## Features
- GPIO setup and configuration
- LED control (Red, Green, Yellow, Blue)
- Basic input handling and state switching
- Written in C and Assembly

## Getting Started
1. Install [PlatformIO](https://platformio.org/)
2. Clone this repository
3. Open the project in PlatformIO-compatible IDE (e.g., VS Code)
4. Build and upload to the HiFive1 Rev B board

## Target Hardware
- SiFive HiFive1 Rev B (RISC-V)

## File Structure
- `src/` - Source code (C and Assembly)
- `include/` - Header files
- `lib/` - Additional libraries
- `platformio.ini` - PlatformIO configuration
